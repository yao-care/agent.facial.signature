# 管理介面操作指南（admin.html）

給機構管理者／維運人員的操作參考。`/`（index）會自動轉到 admin。

## 第一次使用流程

1. 開 `admin.html` → 「**系統工具**」確認 persistent storage 已授權（見 [`operations.md`](operations.md)）。
2. 若要警示：到「**警示名單**」選類型（高風險走失／失智長者／黑名單）→ 點「建立名單」啟用。
3. 開 `example-checkin.html`：選「開啟語音播報」或「不用」→ 允許相機 → 對鏡頭 → 等圓圈轉一圈 → 自動建檔。
4. 回 `admin.html` →「**人員**」tab 為新建檔的人命名（改 input → 點儲存）。
5. 要記電話／關係／聯絡人：點該人員「備註」欄打開編輯器，按 + 新增欄位。
6. 再次入鏡 → 應認得 + TTS 播報「{姓名} {時段問候}」。

## 各 tab

### 人員 tab
- 依「最後簽到」排序；「近 3 日簽到」欄顯示「5/25 10 次 / 5/24 2 次…」（0 次不顯示），點按鈕跳到紀錄 tab 並篩該人。
- 姓名 inline 編輯（改後按「儲存」）。備註欄開 modal 自由新增 key-value（電話、關係、緊急聯絡人、地址、年齡、性別…）；新人建檔時系統自動填「年齡」「性別」（Human library 預測）。
- 操作：合併 / 拆分 / 刪除。篩選：未命名／已命名、辨識模型版本（單版本時隱藏）、搜尋姓名／備註。

### 紀錄 tab
- 所有簽到／警示 events（最多 200 筆，依時間排序）。
- 篩選：類型（簽到／警示）、結果（命中／新人／模糊／警示命中）、未審模糊區、場合。
- 從人員 tab 跳轉時顯示「正在查看 X 的紀錄」橫幅 + [取消篩選]。
- 模糊區紀錄背景反白；可逐筆指派／建檔／忽略，結局記在 `meta.reviewOutcome`。

### 警示名單 tab
- 固定三類型：**高風險走失**（critical 音效）／**失智長者**（warn 音效）／**黑名單**（critical 音效）。
- 每種名單先點「建立名單」啟用；加成員用下拉選人員（不用打 ID）。
- 其他身份分類（家屬／員工／志工／重要訪客／訪客）在「人員」tab 的「身份」欄編輯，不觸發警示音效。

### 情境設定 tab
- 時段週排程（週一→週日 agenda）；報到時依時間自動套用對應的服務項目／活動編號。特定日期優先於星期。

### 服務紀錄報表 tab（社會局 B 表）
- 18 欄、民國年；簽到取當日最早、簽退取最晚；篩選欄即時更新（無「產生報表」按鈕）。
- 最左「登錄」欄勾選 → 變灰 + 「是否平台已登錄」填 Y，存 IndexedDB。
- 選取複製後直接貼回社會局平台。

### 校準參數 tab
- 4 個分組卡片：採樣／比對／容量／系統。數字參數可改；資料庫版本唯讀；按鈕 sticky 在底部。
- 含退冊生物特徵保留期、匯出提醒門檻等可校準參數。

### 系統工具 tab
- **左欄**：儲存狀態（持久儲存、用量）+ 清理孤兒快照 + 備份／還原（匯出支援加密；匯出明文有防呆 confirm）+ 退冊生物特徵清除（預覽／手動／稽核）+ 距上次匯出天數提醒。
- **右欄**：相似度測試器——挑兩位人員，顯示照片 + 算最高 cosine + 自動判斷「視為同一人／不同人／模糊區」。
