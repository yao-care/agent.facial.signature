# 長者友善無接觸報到

一套**用人臉自動報到**的系統：長者只要走到鏡頭前，系統就認出他、語音問候、記下這一次的到場——不用按鈕、不用觸控、不用簽名。

所有資料只留在這台裝置裡，不上雲、不外傳。

**線上服務**：https://sign.yao.care/

---

## 這是給誰用的

- 長照機構、日照中心
- 社區照顧關懷據點
- 任何需要「每天有一群人來、要記錄誰來了」的單位（也可用在員工出勤、課程點名、活動簽到）

## 它解決什麼問題

傳統紙本簽到對長者不友善：手抖、看不清、忘記簽、排隊塞在門口。工作人員事後還要把名單一筆筆key進政府平台。

這套系統把「報到」變成**走過去就好**：

- 長者站到鏡頭前 → 系統自動認出 → 語音說「○○○早安」→ 完成報到
- 不認識的新面孔 → 自動建一筆，工作人員事後補上名字
- 一整天下來，系統自動整理好一份到場清單，工作人員直接複製貼回政府平台

---

## 四大特色

### 👋 無接觸自動報到
長者走到鏡頭前就自動完成，不必按任何按鈕、不必碰螢幕。系統會用語音叫出名字問好（早安／午安／晚安）。

### 🔒 資料只留在本機
人臉特徵、報到紀錄、照片，全部只存在這台裝置的瀏覽器裡，**不上雲端、不傳出去**。沒有外部伺服器能看到這些資料。

### ⚠️ 名單分級警示
可以設定特別關注的名單——例如**高風險走失**、**失智長者**、**黑名單**——當這些人出現在鏡頭前，系統會用不同顏色和提示音主動通知工作人員。

### 📊 報表自動整理
系統把每天的報到自動整理成一份清單，欄位對齊政府社會局平台的格式，工作人員選取後直接複製、貼回平台即可，不用一筆筆重打。

## 兩種使用模式

- **簽到模式** — 偵測到臉 → 認出是誰（或建一筆新的）→ 記下這一次到場。用於各種報到、點名、出勤。
- **警示監看模式** — 偵測到臉 → 比對你設定的關注名單 → 命中就主動跳警示提醒工作人員。用於走失預警、黑名單監看。

---

## 跟政府社會局平台的關係

這套系統的定位是政府平台的**報到自動化前端**，只負責「把報到這件事自動化」：

- 系統會把每天的報到整理成一份**對齊社會局平台格式的清單**，工作人員選取後複製、貼回平台。
- 身分證字號、出生年月日、行政區、里別這些敏感個人資料**留在社會局平台、不進入本系統**，以降低資料外洩風險。
- 活動管理、月報統計這類功能由社會局平台本身負責，本系統不重複做。

## 使用前一定要知道

- **資料只在這台裝置**：清掉瀏覽器資料、換裝置、重灌系統，資料就會不見。請養成**定期匯出備份**的習慣。
- **人臉辨識會出錯**：再好的辨識都可能認錯人或把同一個人當成兩個人。系統內建合併、校正工具讓管理者隨時修正，請定期檢查。

## 法律提醒（白話版）

- **人臉是敏感個資**，跟指紋、DNA 同等級，法律保護比一般姓名電話更嚴。
- **用這套系統的是使用單位（機構／店家），不是工程師或平台**。出事的法律責任在使用單位身上。
- **拍誰的臉，要先讓他知道並同意**。建議在入口貼告示；建檔時系統會跳出「是否同意」的對話框讓對方確認。
- **長者意識不清**要家屬同意；**未成年**要監護人同意；**員工**建議在到職契約裡寫清楚。
- 不照規矩做：台灣個資法可罰 5～50 萬甚至刑責；歐洲 GDPR 罰款上限是年營收的 4%。
- 本系統不負責使用單位的法律合規，使用單位須自行確認符合當地法規（台灣個資法／GDPR／HIPAA 等）。
